# Welcome to Our First Annual Kentucky Derby Bash Workshop

Geat ready for the Run of the Roses!

This README will guide you through the steps necessary to prepare for this workshop.

## 1) Request a High Performance Computing Account: 

Please sign up for a High Performance Computing account using this [link](https://dashboard.dartmouth.edu/research/hpc_account) if you do not already have an account.

## 2) Install and Activate Dartmouth's VPN

Please install the Dartmouth VPN if you do not already have it installed using these [instructions](https://services.dartmouth.edu/TDClient/1806/Portal/KB/ArticleDet?ID=72204). Make sure your VPN is connected prior to signing into Discovery during our workshop, especially if you are off campus.

Please reach out to Noelle.Noreen.Kosarek@dartmouth.edu if you encounter any issues during setup.
